Subtype Constraints

  • We can define a futher constraint which allows one TIP to only contain types present in a second TIP. We use the compliment of HFreeType, HBoundType to define the HSubType constraint.

  • Types can occur only once in each TIP and order is irrelevant.

  • This gives us effective subtypes.

  • Using HSubType x y, HSubType y x defines type equivalence for TIPs.
