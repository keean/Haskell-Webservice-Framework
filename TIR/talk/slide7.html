Functions on HLists ...

  • Basic List operations can be easily defined:
    hHead :: HCons e l -> e
    hTail :: HCons e l -> l
    However we find we have to always be explicit with the types:
    myFn :: HList l => (HCons a (HCons b (HCons c l))) -> c
    myFn l = hHead (hTail (hTail l))
  • hInit and hLast, cannot be defined as ordinary polymorphic functions as we cannot determine the return type.
    hLast :: HCons e l -> ?
