... Functions on HLists

  • We require choice on each recursion based on type, this suggests the following:
    class HLast l e | l -> e where
       hLast :: l -> e
    instance HLast (HCons e HNil) e where
       hLast (HCons e HNil) = e
    instance HLast l e => HLast (HCons e' l) e where
       hLast (HCons _ l) = hLast l
  • Most of the common list operations can be defined with the above technique, requiring a class per function:
    hHead,hTail,hInit,hLast,hAppend etc...
  • See paper for more definitions, including foldr for HLists.
