Ensuring HLists are Well Formed

  • We are dealing with the class of all nested right-associative binary products:
    class HList l
    instance HList HNil
    instance HList l => HList (HCons e l)
  • We can improve on the notation by using infixr operators:
    type MyHList = Key :*: Name :*: Breed :*: Price :*: HNil

    myHList :: MyHList
    myHList = Key 13 .*. Name "Flossy" .*. Cow .*. Price 55.97 .*. HNil
  • We can constrain the .*. operator:
    (.*.) :: HList l => e -> l -> HCons e l
