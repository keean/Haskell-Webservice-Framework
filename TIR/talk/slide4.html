Types of Heterogeneous Collection

  • Strings, using read/show - no typing at all.

  • Sum types with labels - run time typing.
    data A = A Int | B String | C ...
  • Dynamics, the same as Sum types, but the compiler generates a universal sum type for you.

  • None of these collection types provide any static typing, so programs more likely to have run time errors.
