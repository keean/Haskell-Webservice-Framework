Heterogeneous Arrays

  • Using natural numbers at the type level (Peano numbers) we can do indexed access on an HList:
    data HZero = HZero
    data HSucc n = HSucc n

    class HLookup l i e | l i -> e where
       hLookup :: l -> i -> e
    instance HLookup (HCons e l) HZero e where
       hLookup (HCons e _) _ = e
    instance HLookup l i e => HLookup (HCons e' l) (HSucc i) e where
       hLookup (HCons _ l) (HSucc i) = hLookup l i
  • We can define various index based functions:
    hLookup, hDelete, hUpdateTP, hUpdateTC, hProject, hSplit, hSize
  • hProject & hSplit both take an HList as their second argument as well as the first. (See paper for definitions).
