... Type Driven Operations

  • We can also lift the array operations given earlier to the type level.

  • We use a proxy to pass the type for the operation, to make explicit that the value is not used, and to avoid using `undefined'
    data HProxy t = HProxy
  • An example of this lifting is:
    instance (HType2HNat l p i,HDelete l i l')
          => HDelete l (HProxy e) l' where

       hDelete l p = hDelete l (hType2HNat l p)
  • We find that we require both overlapping instances and fundeps to implement hType2HNat. It is possible to implement this in Haskell (with multi-parameter type classes and fundeps) - we will return to this later.
