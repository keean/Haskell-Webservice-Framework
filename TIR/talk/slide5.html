Typeful Heterogeneous Lists

  • Tuples (,,,) although heterogeneous cannot be indexed.

  • Using Peano numbers (type level naturals), we could construct a function to access the Nth member, but that class would nead one instance for each possible tuple length.

  • Solution: use a recursive definition based on right-associative binary products
    (Key,(Name,(Breed,(Price,()))))
  • Because we are interested in static typing we will use HCons and HNil instead:
    data HNil = HNil
    data HCons e l => HCons e l
    resulting in lists like:
    HCons Key (HCons Name (HCons Breed (HCons Price HNil)))
